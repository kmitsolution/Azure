# Exam AZ-104: Microsoft Azure Administrator
Below are the main responsibility of an Azure Administraor
1. Azure administrator's reponsibilty is to implement an organization's cloud infrastructure.
2. Azure administrator's also cordinate with other teams to deliver Azure networking, application devlopement, networking, DevOps solutions, storage etc.

## Prerequisites
<ul>
  <li> Basic Knowledge (No Knowledge for this tutorial) of Azure fundamental. </li>
  <li> Basic knowledge of OS (Windows/Linux). </li>
  <li> Very Basic knowledge of networking. </li>
  <li> Very Basic knowledge of file system. </li>
</ul>  
 
## Skills measured
<ul>
  <li> Manage Azure identities and governance (15–20%) </li>
  <li> Implement and manage storage (15–20%)</li>
  <li> Deploy and manage Azure compute resources (20–25%)</li>
  <li> Configure and manage virtual networking (20-25%)</li>
  <li> Monitor and maintain Azure resources (10–15%)</li>
</ul>
